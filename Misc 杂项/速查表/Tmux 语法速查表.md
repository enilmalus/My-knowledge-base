# Tmux 语法速查表

| 命令     | 解释                           |
| ------ | ---------------------------- |
| tmux   | 进入tmux窗口                     |
| crb c  | 创建新窗口                        |
| crb n  | 切换窗口为n<br>（n 为数字，可在下方查看现在为几） |
| crb %  | 水平分屏                         |
| crb "  | 垂直分屏                         |
| crb d  | 结束当前分屏进程                     |
| crb -> | 切换分屏，->代表有箭头<br>（切换上下左分屏同理）  |
| crb [  | 切换上下浏览模式<br>（按 q 结束上下浏览）     |

> Ctrl + B 为基本控制单位，在 Tmux 中在执行命令前需要先使用 Ctrl + B 在表格中简称为 crb
