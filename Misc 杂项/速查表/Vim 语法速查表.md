# Vim 语法速查表


## 光标移动

| Key | 解释    |
| :--- :|: -----: |
| h   | 向左    |
| k   | 向上    |
| j   | 向下    |
| l   | 向右    |
| w   | 下一个单词 |
| b   | 上一个单词 |

## 编辑文本

| Key |    解释        |
| :-: | :----------: |
|  i  |   在光标之前插入    |
|  a  |   在光标之后插入    |
|  I  |    在行开头插入    |
|  A  |    在行末尾插入    |
| Esc |   进入可视化模式    |
|  d  |      删除      |
| dd  |     删除行      |
|  c  | 删除，然后进入插入模式  |
| cc  | 删除行，然后进入插入模式 |
| yy  |     复制一行     |
|  p  |    在光标后粘贴    |
|  P  |    在光标前粘贴    |
|  x  |    删除当前字符    |
|  X  |   删除上一个字符    |

## 退出

| Key |    解释         |
| - | ----------- |
| :w  |      保存       |
| :wq |     保存退出      |
| :q  |      退出       |
| :q! |     强制退出      |
