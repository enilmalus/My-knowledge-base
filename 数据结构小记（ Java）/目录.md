### 数组

-  优点
	
	1. 存取速度快

-  缺点
	
	 1. 必须事先知道数组的长度
	 2. 插入删除元素很慢
	 3. 空间通常是有限制的
	 4. 需要大块连续的内存块
	 5. 插入删除元素效率很低
